# GitHub Upload Instructions

## Your repository is ready to push!

### After creating a new repository on GitHub, use ONE of these commands:

#### If using HTTPS:
```bash
git remote add origin https://github.com/YOUR_USERNAME/bajaj-finserv-demo.git
git branch -M main
git push -u origin main
```

#### If using SSH:
```bash
git remote add origin git@github.com:YOUR_USERNAME/bajaj-finserv-demo.git
git branch -M main
git push -u origin main
```

### Replace YOUR_USERNAME with your actual GitHub username!
